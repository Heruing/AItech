## Python Overview

- **Python의 특징**
  - 플랫폼 독립적
    - 운영체제에 상관없이 작성하여 사용가능

  - 인터프리터 언어
    - 소스코드를 실행시점에 해석, 간단한 작성, 적은 메모리, 느림
    - 컴파일러 언어: 기계어로 먼저 번역, 빠름, 많은 기억장소 필요

  - 객체 지향
    - 단위 모듈을 중심으로 프로그램을 작성
    - 객체는 method와 attribute를 가지고 있음

  - 동적 타이핑 언어
    - 프로그램 실행 시점에 데이터 타입 결정


- **Why Python?**
  - 쉽고 간단하며 사용법
  - 이해하기 쉬운 문법
  - 다양한 라이브러리
